1) INICIALIZACION
Para Inicializar un proyecto en Node.Js:
	
	npm init --yes

Esto nos crear� el archivo "package.json" con la Descripcion del Proyecto. EJ:

{
  "name": "Bootstrap-template-init",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC"
}
====================================================================================================================================
2) INSTALAMOS LAS DEPENDENCIAS
Luego ejecutamos el siguiente comando para instalar:

* Bootstrap (El Framework de Css)
* Jquery (Lo descargamos por que Bootstrap hace uso de este para poder agregar mas Animaciones)
* Popper.js (Jquery Hace uso de este para las ventanas modales, alertas, acoordeones, etc)
* Font-Awesome (De este obtenemos los Iconos)


	npm install bootstrap font-awesome jquery popper.js

Esto nos actualizara el archivo "package.json" y quedaria asi:

{
  "name": "Bootstrap-template-init",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC",
  "dependencies": {
    "bootstrap": "^4.3.1",
    "font-awesome": "^4.7.0",
    "jquery": "^3.4.1",
    "popper.js": "^1.15.0"
  }
}

Tambien se crear� un directorio con el nombre "node_modules" con los siguientes directorios dentro:

* bootstrap 
* font-awesome 
* jquery 
* popper.js 

====================================================================================================================================
3) INSTALAMOS LAS HERRAMIENTAS QUE NOS AYUDARÁN EN EL DESARROLLO

Luego ejecutamos el siguiente comando para instalar:
* Gulp ( Un Task Runner)
* Herramients de Gulp ( Funciones de Gulp que nos ayudan a ejecutar Tareas Automaticamente )
* Browser-sync ( Generador de un servidor, este nos ayudará a que cada ves que guardemos el proyecto el navegador se refresque )

    npm install -D gulp gulp-cli gulp-sass browser-sync

Como podemos ver despues de "install" agregamos "-D" esto es para que no nos instale estas herramientas como herramientas de dependencias,
ya que nuestro proyecto no dependera de estos, por que solo nos ayudarán en el desarrollo.
El archivo "package.json" se actualizará y quedaria asi:


    {
  "name": "Bootstrap-template-init",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC",
  "dependencies": {
    "bootstrap": "^4.3.1",
    "font-awesome": "^4.7.0",
    "jquery": "^3.4.1",
    "popper.js": "^1.15.0"
  },
  "devDependencies": {
    "browser-sync": "^2.26.5",
    "gulp": "^4.0.2",
    "gulp-cli": "^2.2.0",
    "gulp-sass": "^4.0.2"
  }
}



====================================================================================================================================
4) CREAMOS NUESTROS ARCHIVOS DE TRABAJO

* Creamos el archivo "gulpfile.js" : En este archivo escribiremos todas las tareas automatizadas con Gulp.
* Cremos la siguiente ruta "src/scss/main.scss : Aqui irán todo el codigo Sass que vayamos escribiendo.
* Creamos el archivo HTML en el siguiente directorio "src/index.hmtl: De esta manera vamos creando todos los archivos HTML necesarios

====================================================================================================================================
5) PROGRAMAMOS LAS TAREAS

En el archivo "gulpfile.js" programamos las tareas que queremos automatizar.

====================================================================================================================================
6) CREAMOS NUESTRO COMANDO PERSONALIZADO DESDE EL ENTORNO DE NODE.JS

En el archivo "package.json", dentro del valor "scripts" agregamos:

    "scripts": {
    "start": "gulp"
  }

Nos quedaria de la siguiente forma:

{
  "name": "Bootstrap-template-init",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "start": "gulp"
  },
  "keywords": [],
  "author": "",
  "license": "ISC",
  "dependencies": {
    "bootstrap": "^4.3.1",
    "font-awesome": "^4.7.0",
    "jquery": "^3.4.1",
    "popper.js": "^1.15.0"
  },
  "devDependencies": {
    "browser-sync": "^2.26.5",
    "gulp": "^4.0.2",
    "gulp-cli": "^2.2.0",
    "gulp-sass": "^4.0.2"
  }
}

====================================================================================================================================
7) EJECUTAMOS EL COMANDO PERSONALIZADO QUE SE CREAMOS

Ejecutar el siguiente comando será lo mismo que ejecutar el comando "gulp":

    npm start
